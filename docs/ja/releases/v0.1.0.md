## ハイライト

- Gemini Live のネイティブ音声を活用し、トランスクリプトと音声を同時に進行ベースで再生。
- 16-bit PCM チャンク再生により、応答完了前でも唇同期を開始。
- 同梱 `public/Kiyoka.vrm`、ライブ設定、音声設定、system prompt を含む基本操作が揃った UI を提供。
- `Settings` → `Streaming` → `YouTube relay` で受信コメント連携を有効化し、最新コメントをアプリ内チャットへ反映（受信コメントに限定）。
